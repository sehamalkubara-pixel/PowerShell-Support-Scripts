# PowerShell Support Scripts

This project contains practical PowerShell scripts used in daily IT support tasks.

## Scripts Included

### 1. `check-disk.ps1`
Displays all available file system drives and free space — useful for troubleshooting disk capacity issues.

### 2. `list-users.ps1`
Lists all local user accounts on a Windows system — useful for inventory and user management.

## Usage Instructions

To run a script in PowerShell:

```powershell
.\check-disk.ps1
